# reactclicker
A cookie clicker clone based in ReactJS, as a sort-of motivation to learn ReactJS.

[Demo](http://palomoduarte.com/reactclicker/)

## Component structure

GameWindow

* CookieButton
* PowerUpStore
 * Grandmas
 * Farm
* Messages
